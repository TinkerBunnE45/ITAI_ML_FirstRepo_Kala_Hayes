# ITAI_ML_FirstRepo_Kala_Hayes

## Overview
This repository was created as part of the ITAI-1371-Intro to Machine Learning to introduce students to GitHub and team collaboration.

## Objective
The purpose of this repository is to help students become familiar with GitHub and team collaboration. Through this assignment, students learn how to create repositories, upload and edit shared files, invite collaborators, create pull requests, and merge changes within a shared project.

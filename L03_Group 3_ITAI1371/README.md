### **Overview**
This repository was created for the **Module 3 Lab** in **ITAI-1371-Intro to Machine Learning** to practice building, evaluating, and reflecting on a foundational machine learning workflow.

### **Objective**
The purpose of this repository is to demonstrate our team's understanding of **Supervised Learning** and the **6-step ML workflow**. Through this lab, we learned to move beyond the "syntax wall" of Scikit-Learn to perform exploratory data analysis, train predictive models on the UCI Wine dataset, and distinguish between simple "memorization" and actual model "learning" using a train-test split.

### **Files Included**
*   **Module_3_Lab.ipynb**: The primary Jupyter Notebook containing our data cleaning, visualization (heatmaps), model training, and performance metrics.
*   **Team_Reflection.md**: A deep-dive journal documenting our breakthroughs with the "answer key" concept and our analysis of algorithm performance.

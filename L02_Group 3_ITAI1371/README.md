## 🛠 Tech Stack
We utilized the standard Python data science stack to process the Iris dataset:
*   **Pandas:** For loading the dataset into structured DataFrames.
*   **NumPy:** For mathematical calculations (mean, standard deviation).
*   **Matplotlib:** For rendering scatter plots and bar charts to identify data patterns.

## 📊 Key Findings
*   **Setosa Identification:** Our scatter plot showed that *Iris setosa* has the largest average sepal width and forms a distinct cluster, meaning it can be identified by sepal measurements alone.
*   **Feature Selection:** *Versicolor* and *Virginica* overlapped significantly, indicating that sepal measurements are insufficient to distinguish them—highlighting the importance of additional features like petal dimensions.
*   **Data Integrity:** Bar charts confirmed a perfectly balanced dataset (50 samples per species), which is essential to prevent model bias during training.
